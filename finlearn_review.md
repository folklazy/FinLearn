## 🐛 บั๊กที่พบ (ยืนยันแล้ว)

> [!CAUTION]
> ### 1. `sd.peRatio` แสดงชื่อ variable แทน label
> ในส่วน "ตัวเลขสำคัญ → ความคุ้มค่า" ตัวแรกแสดง **`sd.peRatio`** แทน "P/E Ratio" ขณะที่ตัวถัดไปแสดง "P/B Ratio" ถูกต้อง → แสดงว่า map label ไม่ครบ

![บั๊ก sd.peRatio ในส่วน Key Metrics](C:\Users\Natthachon\.gemini\antigravity\brain\e7868232-47f9-4ef3-aff3-bb33300898db\.system_generated\click_feedback\click_feedback_1773592420995.png)

> [!CAUTION]
> ### 2. Footer Links ทั้งหมดเป็น placeholder (#)
> ลิงก์ใน footer (**คำศัพท์การลงทุน**, **คำถามที่พบบ่อย**, **เกี่ยวกับเรา**) ทั้งหมดยังเป็น `#` ไม่ไปหน้าจริง → สร้างความน่าเชื่อถือลดลง

> [!WARNING]
> ### 3. Label ภาษาปนกัน (Inconsistent Localization)
> ในหน้า Key Metrics บาง label เป็นไทย (รายได้ต่อปี, กำไรสุทธิ) แต่บางอันเป็น EN (Current Ratio, ROE, Profit Margin) → มือใหม่ไทยอาจสับสน

> [!WARNING]
> ### 4. Company Description เป็นภาษาอังกฤษ
> ส่วนสรุปบริษัทบรรทัดแรกเป็นไทย แต่ description ยาวๆ เป็น EN ทั้งหมด → ควรแปลหรือสรุปภาษาไทยเพิ่ม
